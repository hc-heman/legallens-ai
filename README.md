## LegalLens AI ⚖️

GenAI-powered contract risk analyzer for Indian SMEs.

## Features
- Upload business contracts
- Clause-by-clause risk analysis
- Plain-English explanations
- Risk scoring
- Safer clause suggestions

## Tech Stack
- GPT-4 / Claude 3
- Python
- Streamlit

## Run Locally
pip install -r requirements.txt  
streamlit run app.py
